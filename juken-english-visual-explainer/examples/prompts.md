# Prompt Examples / プロンプト例

Use original or fictional prompts when publishing examples. Avoid copying real entrance-exam passages or problem text unless you have permission.

公開例には自作・架空プロンプトを使ってください。実在する入試長文や問題本文の転載は、許諾や引用条件を確認してください。

## Composition Feedback / 英作文添削

```text
次の架空の自由英作文を添削してください。
設問: Do you think students should use digital textbooks at school? Write your opinion in 80-100 words.
答案: I think students should use digital textbooks. Because it is useful and easy to bring. But some students may play games, so teachers should check them.
採点目安、修正理由、改善答案、使い回せる表現を示してください。
```

## Japanese-to-English Translation / 和文英訳

```text
次の架空の和文英訳を解説してください。
「便利だからといって、私たちはいつも新しい技術に頼るべきだとは限らない。」
英訳しやすい日本語への整理、英文の骨格、複数の答案例を示してください。
```

## Sentence Interpretation / 英文解釈

```text
次の架空英文を解釈してください。
The idea that people can learn effectively only when they are interested in the topic is attractive, but it ignores the role of discipline.
主節、従属節、修飾関係を必要に応じて可視化してください。
```

## Reading Comprehension / 長文読解

```text
次の架空の短い英文と設問を解説してください。
Passage: Many cities are encouraging people to use bicycles. This policy reduces traffic and pollution, but it also requires safer roads.
Question: Why do cities support bicycle use?
選択肢がある問題として、根拠の対応と誤答のずれを説明してください。
```

## Grammar and Usage / 文法・語法

```text
次の架空問題を解説してください。
Choose the best answer: If I ( ) more time yesterday, I would have helped you.
A have B had C had had D would have
決め手、時制の考え方、他の選択肢が違う理由を説明してください。
```
