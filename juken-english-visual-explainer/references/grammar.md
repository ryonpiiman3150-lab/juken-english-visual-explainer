# Grammar and Usage Module

Use this module for grammar, vocabulary, usage, error correction, sentence completion, reordering, and transformation tasks.

## Core Goal

Show the deciding rule and why competing choices fail. Make the learner able to reuse the distinction.

## Useful UI Patterns

- Choice selector with live explanation.
- Sentence slot visualization for reordering.
- Contrast table for similar expressions.
- Timeline for tense and aspect.
- Reference arrows for pronouns, relative clauses, and modifiers.
- Compact example switcher for correct and incorrect contrasts.

## Explanation Requirements

- State the deciding clue in the sentence.
- Explain the rule in context.
- Explain why distractors fail.
- Give one or two reusable contrast examples.
- Keep terminology proportionate to the learner's need.

## Avoid

- Listing grammar rules without applying them to the sentence.
- Declaring an answer "unnatural" without explaining the contrast.
- Providing too many examples that obscure the deciding point.
